# RythMe

## Descripción breve

**RythMe** es una aplicación web que funciona como red social centrada en la música. Su objetivo es permitir a los usuarios conocer gente a través de sus gustos musicales mientras descubren nuevos estilos.  

La plataforma permite:  
- Comentar, valorar y debatir sobre canciones o álbumes.  
- Seguir a otros usuarios y descubrir sus gustos musicales.  
- Recomendar nueva música a amigos y contactos.  

RythMe pretende ser un espacio donde la música sea la principal herramienta para conectar personas y compartir experiencias musicales.

---

## Justificación / Motivación

Hoy en día, existen redes sociales para prácticamente todos los hobbies, pero pocas se centran exclusivamente en la música. Aunque plataformas como TikTok o Instagram permiten compartir vídeos con canciones de fondo, no priorizan la música como elemento principal de conexión.  

**RythMe** rompe este paradigma, ofreciendo un espacio donde los usuarios pueden:  
- Conectar de manera auténtica con personas con gustos musicales similares.  
- Compartir sus preferencias musicales y descubrir las de otros.  
- Fomentar el debate y la interacción en torno a canciones y álbumes.  

La música, como lenguaje universal, se convierte en un vehículo para conocer gente y fortalecer relaciones, convirtiendo RythMe en una red social especializada y con un enfoque único.

---

## Objetivos del proyecto

### Objetivo general
Permitir a los usuarios conectar con otras personas a través de sus gustos musicales, fomentando la interacción y el descubrimiento musical.

### Objetivos específicos
- Permitir al usuario valorar canciones o álbumes.  
- Implementar un sistema para seguir a otros usuarios.  
- Permitir recomendar canciones a amigos o contactos.  
- Establecer un sistema de comentarios en cada post para fomentar el debate.
